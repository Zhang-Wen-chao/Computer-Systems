# vim
退格问题
nnoremap <C-?> x 
inoremap <C-?> <BS>
这个配置是因为：
- 你的终端（macOS 通过堡垒机连接）发送的 backspace 键码被识别为 ^?
- vim 默认不知道如何处理这个键码
- 通过重新映射，让 vim 知道遇到这个键码时应该执行什么操作
